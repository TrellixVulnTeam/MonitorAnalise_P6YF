<#lang key="html.sysinfo.description" default=" PRTG requests various system information from your device using standard protocols - WMI and SNMP - and shows the results here. You get system information for Windows devices, running Windows Vista or later, and for Linux, Unix, or other devices that provide SNMP access. Please note that this feature does not fully support all device types. The completeness of your data depends on your system and on the protocols you use.

The following prerequisites are necessary:
  • Credentials for the target system in the Parent or Device Settings
  • WMI enabled on probe and target computer for Windows systems
  • SNMP enabled on the target device
  • Remote Registry Windows service enabled on the target computer
  • Remote Procedure Call (RPC) Windows service enabled on the target computer
Do you need more help? Please see our Knowledge Base: How can PRTG get data for System Information tables?">
