<%NUMBERING1%>.<%NUMBERING2%>.<%NUMBERING3%> PRTG Manual: Template Reload File List

If you have added a new file to the PRTG server while the server was running and the file does not yet appear in this list, please manually reload the file list.

  1. From the main menu bar in the PRTG web interface, choose Setup | System Administration | Administrative Tools.
  2. In section Reload File Lists, click the Go! button.
  3. Your file will now be available in the list.

For more details, see section System Administration—Administrative Tools.
