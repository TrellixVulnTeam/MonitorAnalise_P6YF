PRTG Manual: Application Programming Interface (API) Definition

<%NUMBERING1%>.<%NUMBERING2%>.<%NUMBERING3%> PRTG Manual: Application Programming Interface (API) Definition

The PRTG Application Programming Interface (API) enables you to access monitoring data and manipulate objects using HTTP requests, run your own written sensors and notifications, and implement Mini Probes.

An interactive documentation of the API is also integrated into your PRTG installation: Log in to the web interface and select Setup | PRTG API from the main menu.

PRTG API

The following sections introduce the available options within the PRTG API:

  • HTTP API
  • Live Data
  • Live Graphs
  • Historic Data
  • Object Manipulation
  • Custom Sensors
  • Custom Notifications
  • Mini Probe API

The documentation that comes with your PRTG installation fits exactly the PRTG version you are using.
