Aqui está um aviso do seu servidor PRTG Network Monitor: seu servidor PRTG Network Monitor não pode ser executado em uma das versões mais recentes do Windows Server. Portanto, talvez você não tenha o melhor desempenho possível com o PRTG.


Para uma melhor experiência de monitoramento com o PRTG, recomendamos o Windows Server 2012 R2 ou posterior para o PRTG Core Server Service e o Probe Service.
Para mais detalhes, consulte nossos requisitos de sistema..


Ao mudar para o Windows Server 2012 R2 ou posterior, você pode melhorar especialmente desempenho do monitoramento via WMI até um fator de 10.
Consulte nossa Base de Conhecimento para quais tipos de sensores recomendamos o Windows Server 2012 R2 e por quê.


Obrigado por usar o PRTG!
